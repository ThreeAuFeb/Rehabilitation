#### 项目简介
根据示范动作录入用户视频，并处理视频信息。
#### 知识点
* 使用OpenCV，视频流转换为图片
* 转换视频流为MP4格式的视频文件并保存至本地
#### 启动项目
* 安装node.js,electron
* npm install
* npm install electron-packager --save-dev  //安装打包工具
* npm start  //启动项目
* npm run-script packager  //打包
